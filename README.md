# Vardhan Thadala - Portfolio

A modern, high-performance personal portfolio designed to showcase projects, skills, and professional experience. Built with a focus on immersive UI, fluid animations, and scalable architecture.

## Overview

I am a Full Stack Developer specializing in scalable web applications, modern frontend experiences, backend architectures, and Gen AI integrations. This repository contains the source code for my personal portfolio, reflecting my expertise in building premium, responsive, and robust digital products.

## Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS, GSAP, Framer Motion
- **Backend & Database:** Node.js, FastAPI, PostgreSQL, Supabase, Prisma
- **AI & Data:** Python, LLaMA 3, ChromaDB, RAG Architecture

## Featured Projects

### Nyoraa
A premium branding and creative agency website focused on immersive storytelling, bold visual interactions, and smooth animations. Built with Next.js, Framer Motion, and GSAP.

### Welth
An AI-powered personal finance management platform designed to track expenses, monitor budgets, and generate intelligent financial insights using automated receipt scanning and secure background jobs.

### MEDIMEET
A global doctor appointment and virtual consultation platform enabling patients to connect with healthcare professionals remotely through secure video calls and real-time scheduling.

### Role-Based AI Chatbot
A secure AI-powered internal chatbot system built using Retrieval-Augmented Generation (RAG) and Role-Based Access Control (RBAC), designed to provide department-specific information safely and contextually.

## Features

- **Fluid Animations:** Custom animations powered by GSAP and Framer Motion.
- **Responsive Design:** Fully optimized for seamless experiences across desktop, tablet, and mobile devices.
- **Dynamic Content:** Showcasing a diverse range of projects from e-commerce platforms to AI-integrated enterprise tools.
- **Performance Optimized:** Built on top of Next.js with server-side rendering and optimized asset delivery.

## Local Development

To run this project locally:

1. Clone the repository
2. Install dependencies using the legacy peer dependencies flag
3. Run the development server
4. Open your local server in the browser to view the application
